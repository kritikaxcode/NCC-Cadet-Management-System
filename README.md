# NCC Cadet Management System

A Java-based NCC Cadet Management System developed to manage cadet records, attendance, achievements, and event participation.

## Features

- Add Cadet
- View Cadets
- Search Cadet
- Delete Cadet
- Attendance Management (Planned)
- Achievement Tracking (Planned)
- Event Participation Management (Planned)

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling (Planned)
- Git & GitHub

## Project Status

Currently under development.
